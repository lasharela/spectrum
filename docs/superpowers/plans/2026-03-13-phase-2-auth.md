Update `frontend/lib/shared/widgets/app_text_field.dart`:

```dart
import 'package:flutter/widgets.dart';
import 'package:forui/forui.dart';

/// Builder for prefix/suffix icons matching Forui's FFieldIconBuilder signature.
typedef AppTextFieldIconBuilder = Widget Function(
  BuildContext context,
  FTextFieldStyle style,
  Set<FTextFieldVariant> variants,
);

/// A shared text field widget wrapping Forui's [FTextField].
///
/// Supports label, hint, error display, prefix/suffix icons, password mode,
/// and change callbacks.
class AppTextField extends StatelessWidget {
  final String? label;
  final String? hint;
  final TextEditingController? controller;
  final bool isPassword;
  final bool enabled;
  final ValueChanged<String>? onSubmit;
  final ValueChanged<String>? onChanged;
  final TextInputType? keyboardType;
  final Widget? error;
  final AppTextFieldIconBuilder? prefixBuilder;
  final AppTextFieldIconBuilder? suffixBuilder;

  const AppTextField({
    this.label,
    this.hint,
    this.controller,
    this.isPassword = false,
    this.enabled = true,
    this.onSubmit,
    this.onChanged,
    this.keyboardType,
    this.error,
    this.prefixBuilder,
    this.suffixBuilder,
    super.key,
  });

  @override
  Widget build(BuildContext context) {
    final labelWidget = label != null ? Text(label!) : null;

    if (isPassword) {
      // Don't pass suffixBuilder for password fields — FTextField.password
      // provides its own visibility toggle suffix.
      return FTextField.password(
        control: FTextFieldControl.managed(controller: controller),
        label: labelWidget,
        hint: hint,
        error: error,
        enabled: enabled,
        onSubmit: onSubmit,
        onChange: onChanged,
        keyboardType: keyboardType,
        prefixBuilder: prefixBuilder,
      );
    }

    return FTextField(
      control: FTextFieldControl.managed(controller: controller),
      label: labelWidget,
      hint: hint,
      error: error,
      enabled: enabled,
      onSubmit: onSubmit,
      onChange: onChanged,
      keyboardType: keyboardType,
      prefixBuilder: prefixBuilder,
      suffixBuilder: suffixBuilder,
    );
  }
}
```

> **IMPORTANT — Pseudocode:** The FTextField API parameters (`onChange`, `prefixBuilder`, `suffixBuilder`, `error`) are based on research of the Forui 0.20.0 source. Verify exact parameter names and types against the actual Forui source at `~/.pub-cache/hosted/pub.dev/forui-0.20.0/lib/src/widgets/text_field/`. If the API differs, adapt accordingly. The key concept is correct: FTextField supports error display, prefix/suffix icon builders, and change callbacks.
>
> **Note on `isPassword` + `suffixBuilder`:** `FTextField.password` provides a built-in visibility toggle as its suffix. If you pass a custom `suffixBuilder`, it may override the built-in toggle. When `isPassword` is true, do NOT pass `suffixBuilder` — let Forui handle the visibility toggle. Only pass `prefixBuilder` for password fields.

  /api/auth/forget-password:
    post:
      summary: Request password reset email
      operationId: forgetPassword
      tags: [auth]
      requestBody:
        required: true
        content:
          application/json:
            schema:
              type: object
              required: [email]
              properties:
                email:
                  type: string
                  format: email
      responses:
        '200':
          description: Reset email sent (always returns 200 to prevent email enumeration)
          content:
            application/json:
              schema:
                type: object
                properties:
                  status:
                    type: boolean

  /api/auth/reset-password:
    post:
      summary: Reset password with token
      operationId: resetPassword
      tags: [auth]
      requestBody:
        required: true
        content:
          application/json:
            schema:
              type: object
              required: [token, newPassword]
              properties:
                token:
                  type: string
                newPassword:
                  type: string
                  minLength: 8
      responses:
        '200':
          description: Password reset successful
          content:
            application/json:
              schema:
                type: object
                properties:
                  status:
                    type: boolean
        '400':
          description: Invalid or expired token
